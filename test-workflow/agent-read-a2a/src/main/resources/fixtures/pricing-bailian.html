阿里百炼大模型服务平台 · 模型计费

阿里百炼平台提供通义千问（Qwen）系列模型的 API 服务，以下为截至 2026 年 2 月的定价信息。

Qwen-Max

Qwen-Max 模型输入价格 0.02 元/千 token，输出价格 0.06 元/千 token。 最大上下文窗口 32768 token，默认 QPS 限制 60，支持 Function Calling。

Qwen-Plus

Qwen-Plus 模型输入价格 0.0008 元/千 token，输出价格 0.002 元/千 token。 最大上下文窗口 131072 token，默认 QPS 限制 120，支持 Function Calling。

Qwen-Turbo

Qwen-Turbo 模型输入价格 0.0003 元/千 token，输出价格 0.0006 元/千 token。 最大上下文窗口 8192 token，默认 QPS 限制 120，支持 Function Calling。

特色能力

通义千问系列模型支持 Function Calling、代码解释、长文档理解、多模态理解、 以及 RAG 检索增强。百炼平台提供企业级 API 服务，支持私有化部署与 专属模型微调，适合大规模商业应用。

计费方式

支持按量付费、资源包、和预留实例三种计费方式。新用户可领取免费试用额度， 具体额度以官方活动为准。
