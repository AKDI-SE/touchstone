火山方舟大模型平台提供豆包系列模型的 API 服务，以下为截至 2026 年 3 月的定价信息。

豆包 Pro 4K 模型输入价格 0.0008 元/千 token，输出价格 0.002 元/千 token。 最大上下文窗口 4096 token，默认 QPS 限制 60，支持 Function Calling。

豆包 Pro 128K 模型输入价格 0.008 元/千 token，输出价格 0.024 元/千 token。 最大上下文窗口 131072 token，默认 QPS 限制 30，支持 Function Calling。

特色能力

火山方舟平台支持多模态理解、长文本检索增强（RAG）、Function Calling、 私有化部署、以及企业级安全合规。豆包系列模型在 MMLU、CMMLU、GSM8K 等 基准测试中表现优异，适合企业级应用场景。

计费方式

支持按量付费与资源包两种计费方式。按量付费按实际 token 用量计费； 资源包可预先购买，享受折扣优惠。详细计费规则请参考官方文档。
